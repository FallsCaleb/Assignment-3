  1. HTML
    1. Html is used for structuring the layout of the page.
    2. Html is a markup language that uses a tag system to lable data which can then be altered and stored in a html file.
    3. Html is not a traditional coding language like Java or Python.
  2. CSS
    1. Css is a style sheet language that is used to alter the apperance of the data in an html file.
    2. The style css sheet can take a tag, class, or id and change the height, width, color and more.
    3. Tags have priority in CSS and the vaules given to an object in a lower priority will be overridden by a value given to the same object with a higher priority.
  3. Grid Layout
    1. Grid layout is used to organize the flow of the page
    2. You can section your elements by assigning them to a set area in a matrix.
    3. All specified elements will go into the designated section if those elements are within a tag.
